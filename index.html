CSS Reference

Common CSS properties

Property Name Description
background Sets all background style properties at once, such as color, image, origin and size, or repeat method.
font Sets all the different properties of an element's font.
height Specifies the height of an element.
padding Sets the padding area on all four sides of an element at once.
border Sets an element's border.
margin Sets the margin area on all four sides of an element.

CSS Selectors

Selector Name Description
.class Selects all elements with same class names.
#id Selects the element with the specified id.
* Selects all elements.
